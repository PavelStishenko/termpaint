Termpaint
=========

Termpaint is a low level terminal interface library.

It's designed to be portable and flexible to integrate. It covers event handling and rendering.




It's known to work on
 * xterm
 * vte
 * rxvt-unicode
 * putty
 * konsole
 * linux
 * freebsd
